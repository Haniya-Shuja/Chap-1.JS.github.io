Q1: What is the HTML syntax to add an external JavaScript file?

Q3. How do you write a multiple-line comment in JavaScript?

A3: In JavaScript, you can write multi-line comments using the /* and */ delimiters.
Anything between these delimiters will be treated as a comment and will not be executed by the JavaScript interpreter.

Q4: What is the best way to remove a line of code from running that you might want to keep as you debug?

A4: The best way to temporarily remove a line of code from running in JavaScript, while keeping it for debugging purposes, is to comment out that line or use conditional statements.
